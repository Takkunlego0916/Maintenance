# Maintenance

Maintenance mode plugin for Minecraft servers.

## Features

- Enable / disable maintenance mode
- Automatically kick players without bypass permission
- Server MOTD shows maintenance status
- Automatic language detection based on player client language
- Configurable messages

## Commands

| Command | Description |
|--------|-------------|
| /maintenance on | Enable maintenance mode |
| /maintenance off | Disable maintenance mode |

## Permissions

| Permission | Description |
|-----------|-------------|
| maintenance.admin | Use maintenance commands |

## Configuration

Edit `config.yml` to change messages and MOTD.

## Compatibility

- Minecraft 1.21.1
- Paper / Spigot / Purpur
- Java 21
